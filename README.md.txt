# 🛡 SafeVoice — Secure Workplace Reporting (Prototype)

SafeVoice is a high-fidelity frontend prototype designed to empower women to report workplace harassment safely and anonymously.

## 🚀 Features (Prototype)

* OTP-based onboarding (simulated)
* File incident reports
* Evidence vault with AI verification (simulated)
* Case tracking timeline
* SOS emergency trigger
* Safe journey tracking
* Anonymous support chat

## ⚠️ Disclaimer

This is a **frontend prototype only**:

* No real backend or database
* No real OTP verification
* No real police/SOS integration
* No real encryption or AI processing

All features are simulated for demonstration purposes.

## 🖥 How to Run

1. Download or clone the repo
2. Open `index.html` in your browser

```bash
git clone https://github.com/YOUR_USERNAME/safevoice-app.git
cd safevoice-app
```

Then double-click `index.html`
